# Morpion TD - Console

## Plan de développement :

1. **Initialiser le plateau** - Créer une grille 3x3 (tableau)
2. **Afficher le plateau** - Montrer la grille avec les positions numérotées (1-9)
3. **Gérer les tours** - Alterner entre joueur X et joueur O
4. **Saisir et valider un coup** - Demander une position, vérifier qu'elle est libre
5. **Vérifier la victoire** - Contrôler lignes, colonnes et diagonales
6. **Vérifier le match nul** - Détecter si toutes les cases sont remplies
7. **Boucle de jeu** - Rejouer jusqu'à victoire ou match nul
